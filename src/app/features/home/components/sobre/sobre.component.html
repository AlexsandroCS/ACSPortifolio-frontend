Sobre o site

Inicialmente na criação desse site eu pensei em fazer apenas um portifólio de página única e simples, mostrando as tecnologias que tenho conhecimento e algumas informações sobre minha pessoa, mas é algo que muitas pessoas fazem e também é muito simples. Então eu decidi fazer um blog onde eu postaria todas as minhas anotações de tecnologias que aprendi ou estou aprendendo. Eu também pensei que na construção desse site eu poderia me desafiar implementando funcionalidades mais complexas, pra consolidar ainda mais meu conhecimento e tentar ter uma experiência o mais próximo possível na construção de um projeto.

O site foi construído com as seguintes tecnologias: backend criei um API Rest utilizando o java com Spring Web, Spring Security, JPA e JWT. No frontend, eu optei por utilizar Angular, que era e ainda é uma tecnologia que eu estou estudando, mas nesse projeto eu não utilizei o Angular Material e nenhum framework de CSS, todo HMLT, CSS e Typescript foi escrito por mim mesmo. Também fiz todo o versionamento da construção do site no github, porém estão privados por motivos óbvios.

Para fazer o armazenamento das postagens do site, eu utilizei o MySQL.

Também fiz a integração do TinyMCE, que é uma plataforma para criar conteúdos em um site, fiz seu uso para criar e alterar as postagens no site.
